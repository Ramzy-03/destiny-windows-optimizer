# 🚀 Destiny Optimizer GUI

**Destiny Optimizer GUI** is a powerful Windows performance optimization tool designed to enhance gaming experience by boosting FPS, reducing system latency, and improving overall system responsiveness.

---

## ⚡ How to Run

   - Run Setup.exe
   - Click **"Restart as Administrator"**
   ---

## 🧩 Requirements

- Windows 10 / 11  
- Python 3.10.0 installed ( in the Releases)
- Administrator privileges (recommended)

---

## 🛠️ Fixes 
if you got Error " no module named "tkinter""  
- Fix:
   - Open Terminal (cmd) As admin  
   - type "pip install tkinter"
   - fixed and have fun
     
---

## 🔥 Features

- 🎮 FPS Boost & Gaming Performance Mode  
- 🌐 Network optimization (DNS + TCP tuning)  
- 🧠 RAM cleanup & background process optimization  
- ⚙️ Windows services debloating  
- ⚡ High-performance power plan switching  
- 🧹 System cache cleaning  
- 🧬 Registry-based gaming tweaks  

---

- **FULL BOOST mode applies all optimizations at once.**

- It is recommended to restart your PC after:
  - Network changes  
  - Registry tweaks  
  - Service modifications  
  - Power plan changes  

---

## 🎮 Recommended Usage

- Use **PRO GAMING BOOST** for balanced optimization  
- Use **FULL BOOST** for maximum performance
