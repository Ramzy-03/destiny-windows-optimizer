DESTINY OPTIMIZER GUI

How to run:
1) Extract the ZIP.
2) Double-click Run_GUI.bat.
3) For full permissions, click "Restart as Administrator" inside the app.

Requirements:
- Windows
- Python 3 installed

Notes:
- Some features change Registry, DNS, network stack, power plan, and Windows services.
- Use FULL BOOST only if you really want all actions.
- Restart Windows after network, registry, service, or power-plan changes.
